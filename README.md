# 📝 Todo App (Spring Boot + PostgreSQL)

Учебный pet-проект — REST API для управления списком задач (Todo).
Проект находится в начальной стадии разработки.

---

## 🚀 О проекте

Планируется реализовать backend-приложение на Spring Boot с использованием PostgreSQL для хранения данных.
Основная цель — изучить базовые принципы разработки REST API и работы с базой данных.

---

## 🛠️ Планируемый стек

* Java
* Spring Boot
* Spring Data JPA
* PostgreSQL

---

## 📌 Планируемый функционал

* Создание задач
* Просмотр списка задач
* Получение задачи по ID
* Обновление задач
* Удаление задач

---

## 🎯 Цель проекта

* Разобраться с архитектурой Spring Boot приложения
* Научиться работать с REST API
* Освоить интеграцию с PostgreSQL
* Получить практический опыт backend-разработки
